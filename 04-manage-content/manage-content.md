# Manage Content

_Manage Content_

> Credit-union-wide content management — banner ads, reference data, terms, scheduled notifications, and FICO groups.

The pages in this module are the staff-facing controls for content that appears across all member profiles.

## In this module

* **Banner Ads** — in-app promotional banners.
* **Rates, Static & File Exchange** — reference rates and static content + file exchange.
* **Terms & Conditions** — edit and version T&C documents.
* **Notifications & Downtime** — schedule platform-wide messages.
* **FICO & Groups** — credit-score groupings.
