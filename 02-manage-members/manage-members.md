# Manage Members

_Manage Members_

> The full set of staff actions on a member's profile — find a member, then run support, fraud, compliance, or contact-data actions from one screen.

The pages in this module cover the search flow at the top, then everything reached by drilling into a specific member's profile.

## In this module

* **Search** — find a member by User ID or Member Name.
* **Profile Actions** — Reset Password, Block, Lock, Unenroll, Manage Channels, Sync Contact Details, Update Info New.
* **Overview & Support** — Member Information and Support History.
* **Accounts & Activity** — Accounts, Alerts, Transactions, Activity Log, All Recipients.
* **Devices & Login** — Access Devices, Login Details, Session Details.
* **Business Entitlements** — per-business permissions and limits for commercial signers.
